#  SafeRoads

**Empowering communities to create safer roads through citizen reporting and civic engagement.**

SafeRoads is a modern web application designed to allow citizens to report traffic violations, track safety trends in their community, and assist authorities in identifying high-risk areas. The platform utilizes AI for license plate detection, gamification features to encourage participation, and comprehensive dashboards for data visualization.

##  Features

* **Smart Reporting:** Capture photos of violations directly from your mobile device.
* **AI Integration:** Automatic Number Plate Recognition (ANPR) to detect license plates from uploaded images.
* **Geolocation:** Automatic location tagging (Latitude/Longitude) for accurate mapping.
* **Interactive Dashboards:**
    * **Community Feed:** View recent reports with severity levels and descriptions.
    * **Admin Panel:** A comprehensive control center for authorities to manage reports, users, and violation types.
    * **Analytics:** Charts and graphs visualizing violation trends.
* **Gamification:** Leaderboards to celebrate top contributors and community champions.
* **User Profiles:** Track your own reporting history and manage your account.
* **Responsive Design:** Fully optimized for mobile and desktop experiences.


##  Tech Stack

* **Frontend:** React (TypeScript)
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **UI Components:** shadcn/ui (Radix UI)
* **State Management:** React Query (`@tanstack/react-query`)
* **Routing:** React Router DOM
* **Backend / Database:** Supabase (PostgreSQL, Auth, Storage)
* **Maps:** React Leaflet
* **Charts:** Recharts
* **Icons:** Lucide React
